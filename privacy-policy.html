Privacy Policy

Last updated: January 8, 2026

This Privacy Policy describes how Product Image Downloader ("the Extension") handles user data.

Summary: This extension does NOT collect, store, or transmit any personal user data. All processing happens locally in your browser.

Data Collection

We do not collect any personal information. The extension operates entirely within your browser and does not send any user data to external servers.

Data Storage

The extension stores the following data locally in your browser using Chrome's storage API:

  • User preferences (watermark settings, resize options)
  • Extension settings and configurations

This data is stored only on your device and is never transmitted to any external servers.

Remote Configuration

The extension may fetch a small configuration file from a remote server to retrieve feature settings. During this process:

  • No user data is collected or transmitted
  • Only configuration settings are downloaded
  • The configuration is cached locally

Permissions Used

The extension requires the following permissions:

  • activeTab: To access product images on the currently active webpage
  • downloads: To save images to your computer
  • storage: To save your preferences locally in the browser
  • notifications: To notify you when downloads are complete
  • scripting: To detect and extract images from product pages
  • Host permissions: To access e-commerce websites (Amazon, eBay, etc.) for image extraction

Third-Party Services

This extension does not use any third-party analytics, tracking, or advertising services.

Data Sharing

We do not sell, trade, or otherwise transfer your information to third parties. Since we do not collect any personal data, there is nothing to share.

Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated revision date.

Contact

If you have any questions about this Privacy Policy, please contact us through the Chrome Web Store support page.
